# Linear Systems as Geometry

The equation Ax = b is the central question of linear algebra. You've seen what A does (it transforms space) and what x means (it's the input). But what does it mean to *solve* Ax = b?

It means asking: "what input vector x does the transformation A send to the target b?" There are two ways to *see* the answer.

## Two pictures of the same system

Consider the system:

<MathBlock expression={'\\begin{cases} x + 2y = 5 \\\\ 2x - y = 3 \\end{cases}'} />

**Row picture:** Each equation is a line. The solution is where the lines cross. If the lines are parallel, there's no crossing — no solution.

**Column picture:** Rewrite as x*(1,2) + y*(2,-1) = (5,3). Now it asks: "can I reach the target (5,3) using a linear combination of the columns (1,2) and (2,-1)?"

Toggle between the two views below. Adjust the coefficients and watch what happens when the lines become parallel (det = 0).

<LinearSystemVis client:visible />

## When things go wrong

Three scenarios:
1. **det != 0**: Lines cross at exactly one point. Unique solution. The columns span the whole plane, so any target is reachable.
2. **det = 0, target in column space**: Lines overlap (infinite solutions). The target is reachable, but in many ways.
3. **det = 0, target not in column space**: Lines parallel, no crossing. The target is unreachable — it's not in the column space.

## In three dimensions

In 3D, Ax = b asks whether the target vector b lives in the column space — a plane or line embedded in 3D space.

## Connections

Linear systems are where algebra meets geometry: Ax = b asks "can b be reached by combining A's columns?" The row picture (intersecting lines) and column picture (combining arrows) are two views of the same question. Lesson 20 will formalize the reachable set as the "column space," and lesson 23 will show what to do when b is *unreachable* (least squares).

Gaussian elimination — the algorithm for solving systems — is equivalent to performing row operations that don't change the solution set but simplify the geometry. Each elimination step is a shear or scale that preserves the intersection point.
