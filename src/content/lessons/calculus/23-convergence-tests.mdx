How do you determine whether a series converges? You run through a sequence of tests, each designed to handle different types of series. No single test catches all series. The ratio test is the workhorse for factorials and exponentials. The comparison test is the workhorse for polynomials. The integral test connects series back to areas -- it translates "does the sum converge?" into "does the area under the curve run out?"

## The test flowchart

1. **Divergence test**: Does $a_n \to 0$? If not, the series diverges. Stop.
2. **Is it a p-series?** $\sum 1/n^p$ converges if $p > 1$.
3. **Is it geometric?** $\sum r^n$ converges if $|r| < 1$.
4. **Comparison test**: Compare to a known convergent/divergent series.
5. **Ratio test**: Compute $\lim |a_{n+1}/a_n|$. If $< 1$, converges. If $> 1$, diverges.
6. **Root test**: Compute $\lim \sqrt[n]{|a_n|}$. Same thresholds.
7. **Integral test**: If $f(n) = a_n$ for a positive decreasing $f$, then $\sum a_n$ and $\int_1^\infty f(x)\, dx$ converge or diverge together.

## The ratio test — the workhorse

<MathBlock expression={"L = \\lim_{n \\to \\infty} \\left| \\frac{a_{n+1}}{a_n} \\right|"} />

- $L < 1$: **converges absolutely**
- $L > 1$: **diverges**
- $L = 1$: **inconclusive** (try another test)

The ratio test excels at series with factorials, exponentials, and powers of $n$.

### Example: $\sum n!/n^n$

<MathBlock expression={"\\frac{a_{n+1}}{a_n} = \\frac{(n+1)!/(n+1)^{n+1}}{n!/n^n} = \\frac{n^n}{(n+1)^n} = \\left(\\frac{n}{n+1}\\right)^n = \\left(1 - \\frac{1}{n+1}\\right)^n \\to e^{-1} < 1"} />

Converges. The terms shrink super-exponentially.

## The comparison test

If $0 \leq a_n \leq b_n$ and $\sum b_n$ converges, then $\sum a_n$ converges. Conversely, if $a_n \geq b_n \geq 0$ and $\sum b_n$ diverges, then $\sum a_n$ diverges.

### Example: $\sum 1/(n^2 + n)$

Since $\frac{1}{n^2 + n} < \frac{1}{n^2}$ and $\sum 1/n^2$ converges (p-series with $p = 2 > 1$), the series converges by comparison.

## The integral test

If $f$ is continuous, positive, and decreasing on $[1, \infty)$, then:

<MathBlock expression={"\\sum_{n=1}^{\\infty} a_n \\text{ converges} \\iff \\int_1^{\\infty} f(x)\\, dx \\text{ converges}"} />

This is why the harmonic series diverges: $\int_1^\infty \frac{1}{x}\, dx = \ln x \Big|_1^\infty = \infty$. And $\sum 1/n^2$ converges: $\int_1^\infty \frac{1}{x^2}\, dx = 1$.

<KeyInsight>
No single test handles all series. Start with the divergence test, check for p-series and geometric series, then use comparison, ratio, or integral tests. The ratio test is the most versatile for series with factorials and exponentials.
</KeyInsight>
