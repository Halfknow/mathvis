The derivative is defined as a **limit**. Before we can use that definition, we need to understand what limits are and when they exist.

## What is a limit?

When we write:

<MathBlock expression={"\\lim_{x \\to a} f(x) = L"} />

we mean: as $x$ gets arbitrarily close to $a$ (but never equals $a$), the values of $f(x)$ get arbitrarily close to $L$.

The key word is **arbitrarily close** — not *equal to*, but *approaching*. The function doesn't even need to be defined at $a$.

For over a century, mathematicians used limits intuitively — "it gets closer and closer" — without a precise definition. In 1821, Augustin-Louis Cauchy formalized the limit concept, and Karl Weierstrass later sharpened it into the epsilon-delta definition below. This precision didn't change the geometric intuition; it made it airtight.

## The epsilon-delta picture

The formal definition says: for any tolerance ε > 0 you choose, there exists a δ > 0 such that whenever $|x - a| < δ$, we have $|f(x) - L| < ε$. Think of the limit value $L$ as a target. Epsilon is the size of the bullseye — how close the output needs to land. Delta is how close your input $x$ needs to be to $a$ to hit that bullseye. No matter how small you shrink the bullseye, there's always a neighborhood of $x$-values tight enough to hit it. In the interactive below, the green band is ε and the blue band is δ. ### A complete epsilon-delta proof

Let's prove that <MathInline expression={"\\lim_{x \\to 3} (2x + 1) = 7"} /> from the definition. We must show: for any $\varepsilon > 0$, there exists $\delta > 0$ such that $|x - 3| < \delta$ implies $|2x + 1 - 7| < \varepsilon$.

**Scratch work:** $|2x + 1 - 7| = |2x - 6| = 2|x - 3|$. We want $2|x - 3| < \varepsilon$, so choose $\delta = \varepsilon/2$.

**Proof:** Given $\varepsilon > 0$, let $\delta = \varepsilon/2$. Then whenever $0 < |x - 3| < \delta$:

$<MathInline expression={"|2x + 1 - 7| = 2|x - 3| < 2\\delta = 2 \\cdot \\frac{\\varepsilon}{2} = \\varepsilon \\qquad \\blacksquare"} />$

The structure is always the same: find the relationship between $|f(x) - L|$ and $|x - a|$, then choose $\delta$ to make the final expression fit inside $\varepsilon$.

## The Squeeze Theorem

When a function is trapped between two others that converge to the **same** limit, the trapped function is squeezed to that limit too.

<MathBlock expression={"\\text{If } g(x) \\leq f(x) \\leq h(x) \\text{ near } a, \\text{ and } \\lim_{x \\to a} g(x) = \\lim_{x \\to a} h(x) = L, \\text{ then } \\lim_{x \\to a} f(x) = L"} />

### The fundamental trigonometric limit

The Squeeze Theorem proves the most important trig limit in calculus:

<MathBlock expression={"\\lim_{x \\to 0} \\frac{\\sin x}{x} = 1"} />

The geometric argument: for a small angle $x$ (in radians), draw the unit circle. The area of the triangle with angle $x$ satisfies $\sin x \leq x \leq \tan x$ (the triangle sits inside the sector, which sits inside the larger triangle). Dividing by $\sin x$:

$<MathInline expression={"\\cos x \\leq \\frac{\\sin x}{x} \\leq 1"} />$

As $x \to 0$, $\cos x \to 1$, so both bounds squeeze <MathInline expression={"\\frac{\\sin x}{x}"} /> to $1$.

This result is **not circular** — it's proved geometrically from the unit circle, not from L'Hôpital's rule (which would require knowing the derivative of $\sin x$, which itself requires this limit). We resolve this in lesson 08.

## When does a limit exist?

A limit exists when the function approaches the same value from **both sides**:

<MathBlock expression={"\\lim_{x \\to a^-} f(x) = \\lim_{x \\to a^+} f(x) = L"} />

If the left-hand limit and right-hand limit disagree, the (two-sided) limit does not exist.

Common situations where limits fail:

1. **Jump discontinuity** — the function jumps to a different value (like |x|/x at 0).
2. **Infinite oscillation** — the function oscillates faster and faster near the point (like sin(1/x) at 0).
3. **Vertical asymptote** — the function shoots off to infinity.

## Continuity

A function is **continuous** at $x = a$ if three conditions hold:

1. $f(a)$ is defined
2. <MathInline expression={"\\lim_{x \\to a} f(x)"} /> exists
3. <MathInline expression={"\\lim_{x \\to a} f(x) = f(a)"} />

Informally: you can draw the graph without lifting your pen. Polynomials, exponentials, and trigonometric functions are continuous everywhere on their domains.

## Theorem: The Intermediate Value Theorem (IVT)

If $f$ is continuous on $[a, b]$ and $N$ is any number between $f(a)$ and $f(b)$, then there exists at least one $c$ in $(a, b)$ such that $f(c) = N$.

<MathBlock expression={"f \\text{ continuous on } [a,b],\\ f(a) < N < f(b) \\implies \\exists\\, c \\in (a,b):\\ f(c) = N"} />

Geometrically: a continuous curve cannot jump over a value. If the temperature at noon is 20° and at 3 PM is 25°, then at some moment between noon and 3 PM the temperature was exactly 23° — it had to pass through every intermediate value. The IVT is the reason you can always find a root of a continuous function that changes sign: if $f(a) < 0$ and $f(b) > 0$, then $f$ crosses zero somewhere in between.

The IVT underpins **bisection methods** in numerical analysis — the algorithm that repeatedly halves an interval to locate roots. It is also why binary search works: a sorted array is "continuous" in the discrete sense.

## Why limits matter for derivatives

The derivative is defined as:

<MathBlock expression={"f'(x) = \\lim_{h \\to 0} \\frac{f(x+h) - f(x)}{h}"} />

This limit only makes sense if the function is continuous at $x$. In fact, **differentiability implies continuity** — if $f'(a)$ exists, then $f$ is continuous at $a$. The converse is not true: $|x|$ is continuous at $0$ but not differentiable there (the left and right derivatives disagree).

<KeyInsight>
Limits let us talk about behavior at the boundary — what happens as we approach a point, even if the point itself is undefined or discontinuous. The epsilon-delta picture turns "arbitrarily close" into something precise and visual. The IVT guarantees that continuous functions pass through every intermediate value, and the Squeeze Theorem proves the fundamental trig limit without circular reasoning.
</KeyInsight>

## Connections

The limit is the logical foundation on which the entire derivative machinery rests. In the [previous lesson](/courses/calculus/01-what-is-a-derivative), we defined the derivative as <MathInline expression={"\\lim_{h \\to 0} \\frac{f(x+h)-f(x)}{h}"} /> — here we justified that the limit exists and made the word "approaches" precise. The Squeeze Theorem result <MathInline expression={"\\lim_{x\\to 0} \\sin(x)/x = 1"} /> is essential for deriving the derivatives of trigonometric functions in [lesson 03](/courses/calculus/03-differentiation-rules), and it prevents circular reasoning when we meet L'Hôpital's rule in [lesson 08](/courses/calculus/08-lhopitals-rule).

The IVT connects forward to the **Mean Value Theorem** (a natural extension appearing in curve sketching, lesson 05) and to **root-finding algorithms** used throughout numerical analysis and computer graphics. The idea of "trapping" a value between two bounds reappears in the Squeeze Theorem's cousin: the comparison and ratio tests for series convergence (lesson 23).
