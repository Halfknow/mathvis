You've learned substitution (chain rule backwards) and parts (product rule backwards). Now we add partial fractions -- a technique for rational functions. Together, these three techniques form your integration toolkit. The important thing to understand about partial fractions is that it is algebra, not calculus. The hard part is decomposing the fraction; the integration itself is straightforward (logarithms and arctangents). You do the heavy lifting with algebra, then coast through the calculus.

## The idea

A rational function $\frac{P(x)}{Q(x)}$ can be split into simpler fractions whose denominators are the factors of $Q(x)$. Each simple fraction is easy to integrate.

<MathBlock expression={"\\frac{5x + 3}{(x+1)(x-2)} = \\frac{A}{x+1} + \\frac{B}{x-2}"} />

Solve for $A$ and $B$ by multiplying both sides by $(x+1)(x-2)$ and matching coefficients:

$5x + 3 = A(x-2) + B(x+1)$

Set $x = -1$: $-5 + 3 = A(-3)$, so $A = 2/3$.

Set $x = 2$: $10 + 3 = B(3)$, so $B = 13/3$.

<MathBlock expression={"\\int \\frac{5x+3}{(x+1)(x-2)}\\, dx = \\frac{2}{3}\\ln|x+1| + \\frac{13}{3}\\ln|x-2| + C"} />

## Three cases for partial fractions

### Case 1: Distinct linear factors
$\frac{1}{(x-1)(x+2)} = \frac{A}{x-1} + \frac{B}{x+2}$

### Case 2: Repeated linear factors
$\frac{1}{(x-1)^2(x+2)} = \frac{A}{x-1} + \frac{B}{(x-1)^2} + \frac{C}{x+2}$

Note: you need one term for each power up to the multiplicity.

### Case 3: Irreducible quadratic factors
$\frac{1}{(x^2+1)(x-1)} = \frac{Ax+B}{x^2+1} + \frac{C}{x-1}$

The quadratic factor gets a linear numerator $Ax + B$.

## Integration strategy guide

When you face an integral, work through this checklist:

1. **Simplify** first — can you expand, factor, or cancel?
2. **u-Substitution** — is there an inner function whose derivative appears?
3. **Integration by parts** — is the integrand a product of different function types?
4. **Partial fractions** — is the integrand a rational function?
5. **Trig identities** — for products of trig functions, use half-angle or product-to-sum formulas.
6. **Look it up** — not every integral has an elementary antiderivative.

<KeyInsight>
Partial fractions break a complicated rational function into simple pieces. Each piece integrates to a logarithm, arctangent, or power function. The technique is algebraically intensive but conceptually straightforward — divide and conquer.
</KeyInsight>
