# Scalar Multiplication

If a vector is an arrow, then **multiplying by a number (a scalar)** does exactly one thing: it stretches or shrinks the arrow. The direction either stays the same or flips — it never rotates.

Think of it like this: the scalar is a **stretch factor**. A factor of 2 doubles the length. A factor of 0.5 halves it. A factor of -1 flips the arrow to point the opposite way.

## Stretch, shrink, flip

Drag the scalar slider below — watch the scaled vector stretch, shrink, and flip. The dashed line shows the direction is always preserved.

<ScalarMult client:visible />

Notice what happens when the scalar goes negative: the arrow flips to the exact opposite direction. When it hits zero, everything collapses to the origin. **Scalar multiplication never rotates.**

## In coordinates

When we write it in components, each coordinate gets multiplied:

<MathBlock expression={'c \\cdot \\vec{v} = c \\cdot \\begin{bmatrix} v_1 \\\\ v_2 \\end{bmatrix} = \\begin{bmatrix} c \\cdot v_1 \\\\ c \\cdot v_2 \\end{bmatrix}'} />

The scalar hits every component equally — that's why the direction is preserved. All components stretch by the same factor, so the slope (the ratio of components) doesn't change.

<KeyInsight>
Scalar multiplication is a uniform stretch. It scales length without rotating — or flips direction if the scalar is negative.
</KeyInsight>

## Three special scalars

- **c = 0**: Everything collapses to the origin. The zero vector — no direction, no length.
- **c = 1**: The vector is unchanged. The identity of scaling.
- **c = -1**: The mirror image through the origin. Same length, opposite direction.
