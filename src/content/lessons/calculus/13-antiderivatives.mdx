If the derivative takes you from $F$ to $f$, the antiderivative takes you back from $f$ to $F$. It's the "undo" button for differentiation. Antidifferentiation is like running a film backward -- each derivative rule has a reverse. But there's a twist: the original function could have had any constant added, so you get a whole family of solutions, not just one.

## The definition

$F(x)$ is an **antiderivative** of $f(x)$ if $F'(x) = f(x)$.

Simple enough. But there's a catch: antiderivatives are not unique. If $F(x)$ is an antiderivative, then so is $F(x) + C$ for any constant $C$. Derivatives destroy constant information; antiderivatives can't recover it.

## The family of antiderivatives

The **indefinite integral** captures the whole family:

<MathBlock expression={"\\int f(x)\\, dx = F(x) + C"} />

where $C$ is the "constant of integration" — an arbitrary constant representing the lost information.

Common antiderivatives you should know:

| $f(x)$ | $F(x) + C$ |
|---------|-------------|
| $x^n$ ($n \neq -1$) | $\frac{x^{n+1}}{n+1} + C$ |
| $1/x$ | $\ln|x| + C$ |
| $e^x$ | $e^x + C$ |
| $\sin x$ | $-\cos x + C$ |
| $\cos x$ | $\sin x + C$ |

## Why the constant matters

The constant $C$ isn't just a formality. Consider:

<MathBlock expression={"\\int 2x\\, dx = x^2 + C"} />

If an object's velocity is $v(t) = 2t$, its position is $s(t) = t^2 + C$. The constant $C$ is the starting position. Without it, you know the object accelerates uniformly but you don't know where it started.

In the FTC context, the constant cancels:

<MathBlock expression={"\\int_a^b f(x)\\, dx = [F(x) + C]_a^b = (F(b) + C) - (F(a) + C) = F(b) - F(a)"} />

The constant doesn't affect definite integrals — only the shape of the antiderivative family.

<KeyInsight>
The antiderivative runs the derivative in reverse. Every function has infinitely many antiderivatives, all differing by a constant C. The indefinite integral ∫ f dx = F + C captures the whole family.
</KeyInsight>

<ExerciseBlock number={2}
  prompt="Find F(x) given F'(x) = cos(x) and F(0) = 5."
  hint="The general antiderivative of cos(x) is sin(x) + C. Use the initial condition to find C."
  answer="F(x) = sin(x) + C. F(0) = sin(0) + C = C = 5. So F(x) = sin(x) + 5."
/>
