# Inverse Matrices

If A is a transformation that warps space, then the **inverse** A⁻¹ is the transformation that warps it back. Apply A, then apply A⁻¹, and you're exactly where you started.

<MathBlock expression={'A^{-1} A = A A^{-1} = I'} />

The identity matrix I is the "do nothing" transformation — every vector stays put. So A⁻¹ is the undo button for A.

## When does an inverse exist?

If det(A) ≠ 0, the inverse exists. If det(A) = 0, it doesn't.

Geometrically: if A squashed space flat (det = 0), then multiple points got merged into the same point. You can't un-merge them — the information about which point came from where is gone. There's no way to undo a collapse.

But if A just stretched, rotated, or sheared the space without collapsing it (det ≠ 0), then every point went to a unique destination, and you can trace backwards. The inverse is that backward map.

## Computing the inverse (2x2)

For a 2x2 matrix:

<MathBlock expression={'\\begin{bmatrix} a & b \\\\ c & d \\end{bmatrix}^{-1} = \\frac{1}{ad - bc} \\begin{bmatrix} d & -b \\\\ -c & a \\end{bmatrix}'} />

The pattern: swap the diagonal entries, negate the off-diagonal entries, divide by the determinant. The division by det explains why det = 0 kills the inverse — you'd be dividing by zero.

## What the inverse looks like geometrically

- If A rotates by θ, then A⁻¹ rotates by -θ (rotate back).
- If A scales by 2, then A⁻¹ scales by 1/2 (shrink back).
- If A shears right by 1, then A⁻¹ shears left by 1 (undo the tilt).

Each inverse is the spatial opposite of the original transformation.

<KeyInsight>
The inverse undoes a transformation. It exists precisely when the transformation didn't collapse space — when det ≠ 0. Division by zero isn't just algebraically forbidden; it's geometrically impossible.
</KeyInsight>

## Solving equations

The equation Ax = b asks: "What input x, after transformation A, lands on b?" The answer:

<MathBlock expression={'\\vec{x} = A^{-1} \\vec{b}'} />

If A⁻¹ exists, there's exactly one answer — undo the transformation on b. If A⁻¹ doesn't exist (det = 0), then either there's no solution (b is unreachable) or infinitely many (everything on a collapsed line maps to b).
