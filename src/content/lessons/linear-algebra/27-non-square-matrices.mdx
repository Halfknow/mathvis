# Non-Square Matrices

So far, we've worked with square matrices — transformations that stay in the same dimension. But matrices can be tall, wide, or any shape. A non-square matrix **changes dimensions**.

## Two directions of dimension change

A **wide matrix** (2x3) maps 3D to 2D — it projects or embeds:

<MathBlock expression={'\\begin{bmatrix} 1 & 0 & 0 \\\\ 0 & 1 & 0 \\end{bmatrix} \\begin{bmatrix} x \\\\ y \\\\ z \\end{bmatrix} = \\begin{bmatrix} x \\\\ y \\end{bmatrix}'} />

This 2x3 matrix "flattens" 3D space onto the xy-plane by dropping the z-coordinate. It's a projection.

A **tall matrix** (3x2) maps 2D to 3D — it lifts or embeds:

<MathBlock expression={'\\begin{bmatrix} 1 & 0 \\\\ 0 & 1 \\\\ 0 & 0 \\end{bmatrix} \\begin{bmatrix} x \\\\ y \\end{bmatrix} = \\begin{bmatrix} x \\\\ y \\\\ 0 \\end{bmatrix}'} />

This 3x2 matrix "lifts" 2D vectors into the xy-plane of 3D space. The 2D plane becomes a subspace of 3D.

## The column picture still works

For any matrix (square or not), Ax is still a linear combination of the columns:
- A 2x3 matrix has 3 columns in 2D — the output lives in 2D
- A 3x2 matrix has 2 columns in 3D — the output lives in 3D

The column space is the span of the columns in the output space. Its dimension is the rank.
