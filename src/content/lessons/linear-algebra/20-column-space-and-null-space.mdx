# Column Space and Null Space

Every matrix transformation does two things at once: it **maps inputs to outputs** and it **collapses certain inputs to zero**. The column space describes where things go. The null space describes what disappears. Together, they give a complete picture of the transformation's anatomy.

## Column space — where can you reach?

The **column space** of a matrix A, written Col(A), is the span of its columns. Geometrically: it's all the possible outputs. Every vector Ax is a linear combination of the columns, so the column space is exactly the set of vectors you can reach.

- If the columns span the plane, Col(A) = all of 2D — you can reach anywhere.
- If the columns point in the same direction, Col(A) = a line — you're stuck.
- If the columns are both zero, Col(A) = the zero subspace — everything collapses to the origin.

The dimension of the column space is called the **rank** of A. It counts how many genuinely independent directions the transformation preserves.

## Null space — what gets crushed to zero?

The **null space** of A, written Null(A), is the set of all vectors x such that Ax = 0. Geometrically: it's everything that gets annihilated — mapped to the origin.

- If A is invertible, Null(A) = the zero subspace — only the zero vector disappears.
- If A collapses the plane to a line, Null(A) is a line — everything perpendicular (in some sense) to the surviving direction gets crushed.
- If A sends everything to zero, Null(A) = all of 2D.

## The fundamental trade-off

Here's the deep connection:

> **dim(Col(A)) + dim(Null(A)) = n** (the number of columns)

This is the **Rank-Nullity Theorem**. It says: the dimensions you keep (column space) plus the dimensions you lose (null space) always add up to the total number of input dimensions. Every input dimension is either preserved or destroyed — none are unaccounted for.

**Connections.** Column space and null space are two of the four fundamental subspaces (lesson 22). Together with the row space and left null space, they give a complete anatomical picture of any matrix. The Rank-Nullity Theorem (rank + nullity = n) is one of the deepest results in linear algebra — it says no information is unaccounted for.

## Why this matters for Ax = b

The equation Ax = b has a solution **if and only if b is in the column space** of A. That's tautological but powerful: the column space is the set of reachable destinations. If b is unreachable — outside the column space — then no input x will ever land on b.

And if Null(A) is more than just the zero vector, then when a solution exists, it's not unique. You can add any null space vector to a solution and still have a solution (because A annihilates the null space part).
