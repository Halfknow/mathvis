# Matrix-Vector Product

You know that a matrix's columns are where the basis vectors land. So when you apply the matrix to an *arbitrary* vector (x, y), what happens?

The vector (x, y) is x copies of e1 plus y copies of e2. Since the transformation is linear, the result is x copies of *where e1 lands* plus y copies of *where e2 lands*. In other words:

<MathBlock expression={'A\\vec{x} = x \\cdot (\\text{column 1}) + y \\cdot (\\text{column 2})'} />

The matrix-vector product Ax is a **linear combination of the matrix's columns**. This single insight connects everything you've learned.

## Two ways to compute

The **column picture** (linear combination view):

<MathBlock expression={'\\begin{bmatrix} a & b \\\\ c & d \\end{bmatrix} \\begin{bmatrix} x \\\\ y \\end{bmatrix} = x\\begin{bmatrix} a \\\\ c \\end{bmatrix} + y\\begin{bmatrix} b \\\\ d \\end{bmatrix} = \\begin{bmatrix} ax + by \\\\ cx + dy \\end{bmatrix}'} />

The **row picture** (dot product view): each row of A dots with x to produce one component of the output.

Both give the same answer. The column picture reveals the geometry; the row picture is faster for computation.

## The central equation

This operation — Ax = b — is the central equation of linear algebra. It asks: "can the vector b be written as a linear combination of A's columns?" Everything that follows — solving systems, understanding rank, finding eigenvalues — is a different way of understanding this one equation.

## In three dimensions

The same column-combination view works in 3D. The scaled cube shows how the transformation relocates the entire grid — each output is a combination of the three column vectors.

<FormulaReadout formulas={[

<KeyInsight>
Ax is a linear combination of A's columns, weighted by the entries of x. Read it as "x tells you how much of each column to take."
</KeyInsight>

**Connections.** Ax as a linear combination of columns is the algebraic key to the column space (lesson 20): every possible output Ax is a combination of columns, so the column space IS the set of all Ax. When Ax = b has no solution, b is outside the column space — and least squares (lesson 23) finds the closest reachable point.
